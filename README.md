# [ PREMIUM ] D2 Checkpoint Bot - Sovereign Meta Edition
**Metakure / Metatimes**

A professional, secure automation tool for the Destiny 2 community. Streamline your Raid and Dungeon checkpoint acquisition with our sovereign fleet technology.

## Key Features
* Global Autonomous Fleet: No Twitch login required (System Bots Global) or use your Personal Account.
* * D2Checkpoints.com Integration: Built-in panel for copy-pasting commands from the web.
  * * Tactical UI: Responsive controls, Meta Loading screen, and opacity adjustments.
    * * Secured Architecture: No local tokens; fully encrypted telemetry.
     
      * ## Step-by-Step Operation Guide
     
      * ### 1. Language Selection
      * Currently supporting **English** and **Spanish** interfaces.
     
      * ### 2. Twitch Node Selection
      * Choose the active node for your checkpoint request.
     
      * ### 3. Bungie ID Configuration
      * Enter your Bungie Name (e.g., Name#1234) for automated invite/messaging.
     
      * ### 4. Activity & Checkpoint Selection
      * * Filter by Raid or Dungeon.
        * * Select your desired checkpoint.
          * * Note: Checkpoints marked with (M) indicate Master difficulty.
           
            * ### 5. Dispatch Options
            * * Copy Command: Click the "Copy Command" button to manually paste it into the node's Twitch chat or adjust it as needed.
              * * BOT AUTO SEND:
                *    * Option A (System Bots): Send the request automatically without logging into Twitch (select "System Bots Global" in configuration).
                     *    * Option B (Personal Account): Use "My personal account" to send messages directly using your own Twitch credentials.
                      
                          * ### 6. Side Panel Access
                          * Toggle the "D2CHECKPOINT-BOTS (LIST)" button in the app to access the panel for d2checkpoints.com to copy-paste commands directly.
