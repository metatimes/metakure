# D2 Checkpoint Bot - Sovereign Ultra Premium

![Banner](banner_wide.png)

A professional, secure automation tool for the Destiny 2 community. Streamline your Raid and Dungeon checkpoint acquisition with our sovereign fleet technology.

## Key Features

- **Global Autonomous Fleet**: Request checkpoints automatically using the system's global bots (no Twitch login required) or your own Twitch account.
- - **Integrated Tactical Panel**: Built-in side panel to access and copy/paste join commands directly from `d2checkpoints.com`.
  - - **Intelligent Routing**: Optimized message dispatching to Twitch checkpoint nodes.
    - - **Multilingual Support**: Fully supported in 12 languages.
      - - **Secure & Private**: No personal data collection. Purely tactical automation.
       
        - ## How to Use
       
        - Follow these 4 simple steps to secure your checkpoint:
       
        - 1.  **Select Language**: Choose your preferred language from the list of 12 supported options (EN, ES, FR, DE, IT, PT, JP, ZH, ZHT, PL, RU, KO).
          2.  2.  **Configure Node**: Select the Twitch checkpoint node you wish to use.
              3.  3.  **Bungie ID**: Enter your Bungie ID where you want to receive the checkpoint invitation message.
                  4.  4.  **Target Activity**:
                      5.      - Select the Raid or Dungeon and the specific Checkpoint.
                      6.      - Note: `(M)` indicates Master difficulty.
                      7.      - **Method A (Auto Send)**: Use "BOT AUTO SEND" to let the system handle everything (System Bots Global).
                      8.      - **Method B (Manual/Personal)**: Use "Copy Command" for manual pasting or use "My Personal Account" (requires Twitch login) to send the message yourself.
