# D2 Checkpoint Bot - Sovereign Ultra Premium

![Banner](banner_wide.png)

A professional, secure automation tool for the Destiny 2 community. Streamline your Raid and Dungeon checkpoint acquisition with our sovereign fleet technology.

## Key Features

- **Global Autonomous Fleet**: Request checkpoints automatically using the system's global bots (no Twitch login required) or your own Twitch account.
- - **Integrated Tactical Panel**: Built-in side panel to access and copy/paste join commands directly from d2checkpoints.com. Use the **D2CHECKPOINT-BOTS (LIST)** button to toggle the panel.
  - - **Intelligent Routing**: Optimized message dispatching to Twitch checkpoint nodes.
    - - **Multilingual Support**: Fully supported in 12 languages.
      - - **Secure & Private**: No personal data collection or internal telemetry. Purely tactical automation.
       
        - ## How to Use
       
        - Follow these simple steps:
       
        - 1. **Select Language**: Choose your preferred language from the 12 supported options.
          2. 2. **Configure Node**: Select the Twitch checkpoint node.
             3. 3. **Bungie ID**: Enter your Bungie ID (User#1234) for receiving the invite.
                4. 4. **Target Activity**: Select the Raid/Dungeon. (M) indicates Master difficulty.
                   5. 5. **Send Request**:
                      6.    - **Method A (Auto Send)**: Use "BOT AUTO SEND" for automated delivery via system bots.
                            -    - **Method B (Manual/Personal)**: Use "Copy Command" or "My Personal Account" (Twitch login) to send manually.
