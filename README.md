# [ PREMIUM ] D2 Checkpoint Bot - Sovereign Meta Edition
**Metakure / Metatimes**

A premium, secure tool for the Destiny 2 community, designed to automate and streamline access to Raid and Dungeon checkpoints. This version is the final sovereign rollout, ensuring no sensitive data is exposed and providing a seamless tactical experience.

## Key Features
+ Manual & Automatic Requests: Request checkpoints manually or let the bot handle it automatically across multiple channels.
+ + Autonomous Fleet: Works without the need for a Twitch login (using system global bots) or with your personal Twitch account for direct communication.
  + + D2Checkpoints.com Integration: Integrated side panel to easily browse and copy/paste join commands from d2checkpoints.com.
    + + Tactical UI: "D2CHECKPOINT-BOTS (LIST)" button to toggle the side panel, providing a clean and efficient workspace.
     
      + ## How It Works (Step-by-Step)
      + 1. Language Selection: Choose your preferred language (English and Spanish supported).
        2. 2. Twitch Node Selection: Select the active Twitch node for the request.
           3. 3. Bungie ID Configuration: Enter your Bungie ID (Name#1234) so the bot knows where to send the invite/message.
              4. 4. Activity & Checkpoint Selection:
                 5.    - Choose your desired Raid or Dungeon activity.
                       -    - Select the specific checkpoint.
                            -    - Note: Checkpoints marked with (M) indicate Master difficulty.
                                 - 5. Dispatch Options:
                                   6.    - Copy Command: Click the "Copy Command" button to manually paste it into the node's Twitch chat or adjust it as needed.
                                         -    - BOT AUTO SEND:
                                              -      - Global System Bots: Send the request automatically without logging into Twitch (select "System Bots Global" in configuration).
                                              -       - Personal Account: Use "My personal account" to send messages directly using your own Twitch credentials.
                                          
                                              -   ## Sovereign Security
                                              -   This edition has been hardened for production. No tokens or sensitive data are stored in the codebase. All communication is managed through environment variables and local encryption.
