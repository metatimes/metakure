# 🛡️ D2 Checkpoint Generator - Meta Edition
![Banner](banner_wide.png)

Una herramienta premium y segura para la comunidad de Destiny 2, diseñada para automatizar y agilizar el acceso a checkpoints de Raids y Mazmorras.

## ✨ Características Principales

- **Base de Datos Definitiva**: Incluye todos los encuentros actualizados (Post-Templar, variantes M, etc.).
- **Seguridad Meta Enforced**: El programa está bloqueado a su nombre original para proteger la autoría.
- **Transparencia Personalizable**: Slider integrado para ajustar la opacidad mientras juegas.
- **Panel Web "Clean"**: Interfaz de d2checkpoint.com optimizada y sin publicidad.

## 🚀 Instalación y Uso

1. Descarga el archivo `D2ChkBotMeta.exe`.
2. Ejecuta el archivo (verás la pantalla de carga de Meta Security).
3. Selecciona tu actividad, copia el comando y ¡disfruta!
